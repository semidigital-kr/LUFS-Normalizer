# LUFS Normalizer 🎚️

The perfect tool to uniformly adjust the volume (LUFS) of multiple audio files (.mp3, .wav) to a specific target without quality loss. It prevents digital clipping and preserves 100% of all metadata (album art, ID3 tags, etc.).

- For macOS: [LUFS_Normalizer_macOS_by_wani.zip](https://github.com/semidigital-kr/LUFS-Normalizer/releases/download/v0.0.2/main.py.macOS.build.by.wani.zip)

## 💡 Features

* **Precise 2-Pass LUFS Normalization:** Utilizes the FFmpeg engine to achieve the exact target LUFS without margin of error.
* **Audio Format Flexibility:** Supports both `.mp3` and `.wav` files seamlessly.
* **100% Metadata Preservation:** Original album art, track titles, and artist information remain completely intact after conversion.
* **Intuitive UI:** Features a sleek, dark-mode flat design with a clear progress bar to monitor the process at a glance.
* **Smart Memory Configuration:** Automatically remembers your last custom Target LUFS value.
* **Pure Dynamics & Safety:** No compression/squashing applied; clear visual alerts prevent unexpected digital clipping.
